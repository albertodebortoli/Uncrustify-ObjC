# Uncrustify ObjC

Uncrustify configuration file for Objective-C files. [Uncrustify](https://github.com/bengardner/uncrustify) is a code beautifier tool.

Simple usage:

    ./uncrustify -c ADB_beautifier.cfg -f inputFile.m > outputFile.m
